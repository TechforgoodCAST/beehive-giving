Insights from using open data to provide a grant funding suitability checking tool

Beehive was a free funding suitability checking tool that used a funderâ€™s guidelines, priorities and open data to produce a report (example) that helped fund seekers decide where to apply.

After checking well over 18,000 proposals from individuals and organisations seeking charitable funding Beehive was retired in September 2020.

This site includes an analysis of the proposals submitted to Beehive, key learnings on developing and delivering a tech for good product, as well as some background information about the project. You can find the source code for the project before it was retired on GitHub.

If you have any questions about the project please contact support@beehivegiving.org.
